
# Wtyczka pobierająca dane OSM dla miasta Olsztyn

Prosta wtyczka dla QGIS'a, która umożliwa pobranie m.in warstw:
- Standardowa mapa OpenStreetMap z pełnymi detalami,
- Mapa topograficzna ze szlakiami i warstwicami,
- Mapa z wyróżnionymi ścieżkami rowerowymi,
- Mapa humanitarna z wyróżnionymi budynkami,
- Mapa komunikacji publicznej i transportu,
